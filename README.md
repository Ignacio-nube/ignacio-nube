
# Portfolio Ignacio Márquez

Portfolio personal desarrollado en React + Chakra UI + Framer Motion.

- Presentación profesional y contacto
- Proyectos destacados con imágenes y tags
- Animaciones suaves y diseño glassmorphism

## Instalación

```bash
git clone https://github.com/Ignacio-nube/ignacio-nube.git
cd ignacio-nube
npm install
npm run dev
```
